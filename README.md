# Python Functions Lab

A hands-on practice lab focused on defining and calling functions in Python, manipulating simple data types, and returning values based on specific parameters.

## Exercises Completed

* **Exercise 1: Calculate Area of a Triangle**
    * **Function:** `calculate_area_triangle(base, height)`
    * **Purpose:** Accepts the base and height of a triangle and returns its calculated area as a float.
